# WisdomVault - Server

Backend API for the WisdomVault platform.

It provides authentication support, lesson management, premium membership integration, dashboard analytics, and secure REST APIs using Express.js and MongoDB.

Create a `.env`

```env

DB_NAME=

BETTER_AUTH_SECRET=

BETTER_AUTH_URL=

GOOGLE_CLIENT_ID=

GOOGLE_CLIENT_SECRET=

STRIPE_SECRET_KEY=
```

# 🚀 Run Locally

Clone repository

```bash
git clone https://github.com/mahdihasanprogrammer/Assignment-10-WisdomVault-Server.git
```

Install packages

```bash
npm install
```

Run server

```bash
node index.js
```
